# YmtMD
仰明堂MD — 简洁优雅的 Markdown 编辑器
---

## 一、菜单栏

### 文件菜单

| 菜单项 | 操作内容 |
|--------|----------|
| **新建** (Ctrl+N) | 清空编辑器内容，重置文件名为"未命名.md"。如果当前内容未保存，弹出确认对话框：「不保存 / 取消」 |
| **打开** (Ctrl+O) | 弹出系统文件选择对话框，筛选 Markdown 文件（`.md` / `.markdown` / `.mdown`）。选中后读取文件内容加载到编辑器，标题栏显示文件名 |
| ─ | 分隔线 |
| **保存** (Ctrl+S) | 如果已有文件路径，直接覆盖写入；如果没有（新文档），弹出「另存为」对话框选择保存位置 |
| **另存为...** (Ctrl+Shift+S) | 弹出保存对话框，选择路径和文件名，保存为 `.md` 文件。保存后标题栏更新为新的文件名 |
| ─ | 分隔线 |
| **导出为 PDF** | 将当前文档渲染后的 HTML 保存为 PDF 文件。弹出保存对话框，选择 `.pdf` 路径。使用隐藏 BrowserWindow 渲染 → `printToPDF` → 写入文件 |
| **导出为 HTML** | 将当前文档渲染后的 HTML 包裹完整模板（含 CSS 样式）保存为 `.html` 文件。可在浏览器中直接打开 |
| ─ | 分隔线 |
| **退出** | 关闭程序 |

### 编辑菜单

| 菜单项 | 操作内容 |
|--------|----------|
| **撤销** (Ctrl+Z) | 撤销上一步编辑操作（CodeMirror 内建撤销栈） |
| ─ | 分隔线 |
| **剪切** (Ctrl+X) | 剪切选中的文本到剪贴板 |
| **复制** (Ctrl+C) | 复制选中的文本到剪贴板 |
| **粘贴** (Ctrl+V) | 从剪贴板粘贴文本到光标位置 |
| ─ | 分隔线 |
| **一级标题** | 在光标行首插入 `# `（如果已有标题则替换级别） |
| **二级标题** | 在光标行首插入 `## `（已有则替换） |
| **三级标题** | 在光标行首插入 `### `（已有则替换） |
| **四级标题** | 在光标行首插入 `#### `（已有则替换） |
| ─ | 分隔线 |
| **加粗文字** (Ctrl+B) | 编辑器中有选区时，用 `**text**` 包裹。预览编辑模式中，对选中文本应用 `<b>` 标签 |
| **倾斜文字** (Ctrl+I) | 编辑器中有选区时，用 `*text*` 包裹。预览编辑模式中，应用 `<i>` 标签 |
| ─ | 分隔线 |
| **引用** | 在光标行首插入 `> `（如已是引用则移除，切换式） |
| **列表** | 在光标行首插入 `- `（如已是列表则移除，切换式） |
| **有序列表** | 在光标行首插入 `1. `（如已是有序列表则移除，切换式） |
| **任务列表** | 在光标行首插入 `- [ ] `（如已是任务列表则移除，切换式） |
| ─ | 分隔线 |
| **删除线** | 编辑器中有选区时，用 `~~text~~` 包裹。无选区时在光标处插入 `~~~~`，光标置于中间 |
| **行内代码** | 编辑器中有选区时，用 `` `text` `` 包裹。无选区时在光标处插入 ` `` `，光标置于中间 |
| **分割线** | 在光标所在行**之后**插入新行 `---`。不经过 applyFormat 校验，独立处理 |

### 视图菜单

| 菜单项 | 操作内容 |
|--------|----------|
| **工具栏** (Ctrl+T) | 切换工具栏的显隐。工具栏位于标题栏之上 |
| ─ | 分隔线 |
| **分栏模式** (Ctrl+1) | 源码编辑区与渲染预览区左右并排显示，中间可拖拽分隔栏（比例范围 20%~80%） |
| **编辑模式** (Ctrl+2) | 仅显示源码编辑区，隐藏预览区域 |
| **预览模式** (Ctrl+3) | 仅显示 Markdown 渲染预览，隐藏编辑区域 |
| ─ | 分隔线 |
| **开发者工具** | 打开 Chrome DevTools |
| **重新加载** | 重新加载渲染进程 |

### 帮助菜单

| 菜单项 | 操作内容 |
|--------|----------|
| **关于 仰明堂MD** | 打开独立关于窗口（400×500 不可调整），显示软件名称、版本、说明文字、作者和可点击的网址链接 |

## 二、标题栏按钮

| 按钮 | 操作内容 |
|------|----------|
| 📄 分栏 | 切换到分栏模式（编辑器 + 预览并排），等同于菜单「视图→分栏模式」|
| ✏️ 编辑 | 切换到纯编辑模式，等同于菜单「视图→编辑模式」|
| 👁 预览 | 切换到纯预览模式，等同于菜单「视图→预览模式」|

选中态的按钮背景变为蓝色（`#0078d4`）。

## 三、工具栏按钮（Ctrl+T 切换显隐）

工具栏位于标题栏之上，按功能分组，组间有灰色分隔线。

### 文件操作组

| 按钮 | Tooltip | 操作内容 |
|------|---------|----------|
| 📄 新建 | 新建 (Ctrl+N) | 清空编辑器，同菜单「文件→新建」 |
| 📂 打开 | 打开 (Ctrl+O) | 弹出文件选择对话框，同菜单「文件→打开」 |
| 💾 保存 | 保存 (Ctrl+S) | 保存当前文件，同菜单「文件→保存」 |
| 📁 另存为 | 另存为 (Ctrl+Shift+S) | 弹出另存为对话框，同菜单「文件→另存为」 |
| 📕 PDF | 导出 PDF | 导出为 PDF，同菜单「文件→导出为 PDF」 |
| 🌐 HTML | 导出 HTML | 导出为 HTML，同菜单「文件→导出为 HTML」 |

### 标题组

| 按钮 | Tooltip | 操作内容 |
|------|---------|----------|
| H1 | 一级标题 | 行首插入 `# ` |
| H2 | 二级标题 | 行首插入 `## ` |
| H3 | 三级标题 | 行首插入 `### ` |
| H4 | 四级标题 | 行首插入 `#### ` |

### 文字样式组

| 按钮 | Tooltip | 操作内容 |
|------|---------|----------|
| **B** | 加粗 | 选区包裹 `**text**` 或预览中应用 `<b>` |
| *I* | 倾斜 | 选区包裹 `*text*` 或预览中应用 `<i>` |
| ~~S~~ | 删除线 | 选区包裹 `~~text~~` |

### 列表组

| 按钮 | Tooltip | 操作内容 |
|------|---------|----------|
| ❝ | 引用 | 行首切换 `> ` |
| • | 无序列表 | 行首切换 `- ` |
| 1. | 有序列表 | 行首切换 `1. ` |
| ☐ | 任务列表 | 行首切换 `- [ ] ` |

### 代码/分割线组

| 按钮 | Tooltip | 操作内容 |
|------|---------|----------|
| `</>` | 行内代码 | 选区包裹 `` `text` `` 或光标处插入 ` `` ` |
| — | 分割线 | 在光标行后插入 `---`（独立处理，不经过 applyFormat） |

## 四、预览区操作

### 预览编辑按钮

位于预览区右上角：

| 按钮状态 | 操作内容 |
|----------|----------|
| 🔓 编辑预览 | **点击后**开启预览编辑模式。预览区变为可编辑，出现绿色边框（`#28a745`），修改内容自动同步回源码编辑器 |
| 🔒 关闭编辑 | **点击后**关闭预览编辑模式。预览区恢复只读，源码编辑器重新获得焦点 |

**预览编辑模式下**：
- 输入文字 → 实时自动同步回 CodeMirror（通过 Turndown 将 HTML 转回 Markdown）
- 粘贴内容 → 自动同步
- Ctrl+Z → 转发到编辑器撤销操作
- 右键菜单 → 支持剪切/复制/粘贴/全选

## 五、右键菜单

### 编辑器区右键

| 菜单项 | 出现条件 | 操作内容 |
|--------|----------|----------|
| 剪切 | 有选中文本 | 复制选中文字到剪贴板，删除选中的内容 |
| 复制 | 有选中文本 | 复制选中文字到剪贴板 |
| 粘贴 | 始终出现 | 从剪贴板读取文本，插入到光标位置（优先 clipboard API，降级到 hidden textarea + execCommand） |
| 全选 | 始终出现 | 全选编辑器全部内容 |

### 预览区右键

| 菜单项 | 出现条件 | 操作内容 |
|--------|----------|----------|
| 剪切 | 预览编辑开启 + 有选中文本 | 复制选中文字到剪贴板，删除内容 |
| 复制 | 有选中文本 | 复制选中文字到剪贴板 |
| 粘贴 | 预览编辑开启 | 从剪贴板读取文本，插入到光标位置，自动同步到源码 |
| 全选 | 预览编辑开启 | 全选预览区内容 |

## 六、鼠标拖拽操作

### 分隔栏拖拽

| 操作 | 效果 |
|------|------|
| 鼠标悬停在分隔栏上 | 分隔栏变为蓝色（`#0078d4`） |
| 按住拖拽 | 左右调整编辑区和预览区的宽度比例，最小 20%，最大 80% |
| 释放鼠标 | 比例固定 |

仅限于**分栏模式**下操作。
